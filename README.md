# Portfolio

This is a personal portfolio website built using **React + Vite**.

It showcases my projects, skills, and contact details.  
I have used modern tools like React, Tailwind CSS, and deployed it via Netlify.

---

## Tech Stack

- React
- Vite
- Tailwind CSS
- Git & GitHub
- Netlify
